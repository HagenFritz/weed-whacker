# Ideas
_These are random ideas that could be used to expand the game_

### Random Events
Have random events occur like:
- **Weed Invasion**: All plots grow weeds _immediately_
- **Weather**: Rain slows weed growth, sunny days speed it up
- **Fire**: Random fire starts, killing all weeds

### New Plot Mysteries
When you purchase a new plot, it may have mysteries like:
- **Weed Infestation**: The plot starts with a weed already
- **Find Money**: You find money on the plot
